### For Users
1. **Start Small**: Begin with simple tests
2. **One Variable**: Test one thing at a time
3. **Sample Size**: Wait for statistical significance
4. **Duration**: Run tests for at least 1-2 weeks
5. **Documentation**: Document hypotheses and results

### For Developers
